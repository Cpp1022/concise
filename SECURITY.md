# Security Policy

## Supported Versions

The current `main` branch is supported.

## Reporting a Vulnerability

Please report security issues privately through GitHub Security Advisories if available, or contact the repository owner.

Do not open a public issue for secrets, token exposure, account compromise, or supply-chain concerns.

## Scope

This project is a local Codex skill. The main security risks are:

- accidentally committed secrets
- malicious edits to skill instructions
- misleading install instructions

Review `SKILL.md` carefully before installing or modifying it.
